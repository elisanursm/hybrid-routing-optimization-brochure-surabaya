# Hybrid Optimization for Brochure Distribution Routes in Surabaya

## 📌 Overview
This project focuses on optimizing travel distance for distributing tutoring center brochures to schools across Surabaya. The problem is modeled as a routing optimization problem to improve route efficiency and reduce total travel distance.

## 🎯 Objective
The main objective of this project is to determine efficient distribution routes for brochure delivery in the Surabaya area by minimizing total travel distance.

## ⚙️ Methods
This project compares three hybrid optimization approaches:

- Ant Colony Optimization (ACO)
- Tabu Search + Particle Swarm Optimization (Tabu + PSO)
- 2-opt + Simulated Annealing (2-opt + SA)

These methods are used to evaluate alternative routing strategies and identify the most efficient route configuration.

## 📍 Problem Context
Distributing brochures manually to many schools may lead to inefficient travel routes, higher operational costs, and longer delivery time. Therefore, optimization techniques are used to support route planning for promotional distribution activities.

## 📊 Dataset
The dataset contains distribution points representing schools in Surabaya. These points are used as route nodes for optimization.
